# Al Noor Trust Website

Single-page React website for Al Noor Trust, built with Vite, Tailwind CSS, and Yarn.

## Run locally

```bash
yarn install
yarn dev
```

## Build for production

```bash
yarn build
yarn preview
```

## Key content sections

- Hero and mission summary
- Impact statistics
- About and challenges
- Sponsorship programs
- Donation channels and bank details
- Sponsor-a-child fee table
- Additional support needs
